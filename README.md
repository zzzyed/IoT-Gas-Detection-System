======================= System Introduction =======================

An intelligent gas monitoring system based on the Blynk IoT platform.

The system is built using an ESP32 microcontroller, buzzer, OLED (SSD1306) sensor, MQ2 sensor, Blynk IoT platform, and peripheral hardware components.

====================== Features and Functions ========================

OLED concentration display with animation effects.

Bar chart and line chart status switching.

Customizable system parameters.

Remote data monitoring and control.
