====================================================================================================

基于Blynk IoT平台的智能气体监控系统。
设计使用ESP32、蜂鸣器、OLED(SSD1306)、MQ2传感器、Blynk IoT平台及外围硬件组件构建。

====================================================================================================

特点与功能：

OLED浓度展示，动画效果。柱状图，折线图状态切换。
自定义系统参数。
远程数据监控与控制。
